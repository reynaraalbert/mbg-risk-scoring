# 🍱 MBG Risk Scoring
**Sistem Penilaian Risiko Keracunan — Program Makan Bergizi Gratis**  
Badan Gizi Nasional Republik Indonesia

---

## Apa ini?

Aplikasi dashboard untuk membantu BGN dalam memantau dan menganalisis kejadian keracunan makanan dalam program MBG. Data insiden diunggah, lalu sistem otomatis menghitung tingkat risiko setiap kejadian dan menyajikannya dalam bentuk grafik dan tabel yang mudah dibaca.

---

## Cara Pakai

1. Buka aplikasi di browser
2. Upload file data keracunan (format `.xlsx` atau `.csv`)
3. Sistem otomatis menghitung skor risiko setiap kejadian
4. Lihat hasil analisis, grafik, dan tabel
5. Download laporan jika diperlukan

---

## Tingkatan Risiko

| Level | Skor | Artinya |
|---|---|---|
| 🔴 Kritis | ≥ 75 | Perlu tindakan darurat segera |
| 🟠 Tinggi | 50–74 | Perlu penanganan prioritas |
| 🟡 Sedang | 25–49 | Perlu pemantauan lebih intensif |
| 🟢 Rendah | < 25 | Pemantauan rutin |

---

## Fitur

- Upload file Excel atau CSV
- Deteksi otomatis kolom data
- Perhitungan skor risiko per insiden
- Grafik distribusi risiko, tren bulanan, dan sebaran provinsi
- Filter data berdasarkan wilayah, tahun, dan tingkat risiko
- Tambah, edit, dan hapus data langsung di dashboard
- Download hasil analisis sebagai CSV
